### 가이드

* 실행시 myitem.txt 파일이 없으면 자동 생성
* my item set : 지정한 파일에서 아이템 코드들을 불러와서 얻음
* 저주 제거 : 효과 없는거 같음
* Give set id : 왼쪽 텍스트 박스에 아이템 코드 입력한 아이템음 얻음
* 탄환의 주인 : 앤딩 필수(?) 아이템들 얻음
* item list out : 게임상에 존재하는 아이템 목록을 PickupObjectDatabase.txt 파일로 출력. 근데 도움은 안됨
